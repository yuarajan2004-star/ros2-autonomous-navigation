# AutoNav Bringup

## Purpose

Launches the complete autonomous navigation stack.

## Components

* Gazebo Simulation
* Robot Description
* SLAM Toolbox
* RViz
* ROS-GZ Bridge

## Launch

```bash
ros2 launch autonav_bringup autonav.launch.py
```

## Configuration

Configuration files are stored in:

config/

## Package Role

System orchestration and startup.
