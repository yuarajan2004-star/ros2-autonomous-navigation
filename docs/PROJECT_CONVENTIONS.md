# AutoNav Project Conventions

## Package Naming

autonav_<module>

Examples:

* autonav_localization
* autonav_slam
* autonav_planning

## Topic Naming

Examples:

* /scan
* /odom
* /map
* /cmd_vel

## Frame Naming

Required Frames:

* map
* odom
* base_link

## Launch Files

Store in:

launch/

## Parameter Files

Store in:

config/

## Tests

Store in:

test/
